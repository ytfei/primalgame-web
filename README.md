## 快速开始

```sh
pnpm install
pnpm global add commitizen
pnpm dev
```

本项目使用 `commitlint` 验证 `git commit` 请按照 [**Angular 规范的 Commit Message**](https://i3o61hzm64.feishu.cn/docs/doccnp47z5z4IzGtbv4ZBjom6Od) 提交代码

### 依赖检查

```sh
node v14.17.3
```

```sh
pnpm build
```
